## Venue Information

ARR welcomes position papers. Select the corresponding track and indicate the paper type (technical or position).

## Type 1: Model Hypothesis for a Phenomenon

### Structure
- Introduction
- Model proposal
  - Model component 1
  - Model component 2
  - Model component 3
- Discussions
  - What does this paper mean for future work
- Conclusion

### Key Elements
- There is a hypothesis/thinking model.
- There are case studies describing how the hypothesis fits observations.

## Type 2: A Collection of Different Opinions with a Flat Structure

### Paper Structure
1. Raise many meaningful points.
2. Group them into 3 main sections.
3. For each point (which now turns into subsections), structure three components to make the flow logical.
